# Thermal Classification Datasets

Stratified 60/20/20 split of thermal time-series data from 19 experiment videos (550 tracklets).

## Files
- `final_thermal_train_dataset.csv` -- Training set (330 tracks)
- `final_thermal_validation_dataset.csv` -- Validation set (110 tracks)
- `final_thermal_test_dataset.csv` -- Test set (110 tracks)

## Class Distribution

| Class   | Train | Val | Test | Total |
|---------|-------|-----|------|-------|
| Glass   |    79 |  26 |   27 |   132 |
| Metal   |    79 |  26 |   26 |   131 |
| Paper   |    59 |  20 |   19 |    98 |
| Plastic |   113 |  38 |   38 |   189 |
| **Total** | **330** | **110** | **110** | **550** |

## Format
```
class,uniqueID,track_length,mean_thermal_intensity_array
glass,0797_25,1003,56.7728 56.9523 56.3931 ...
```

- `class`: Material class (glass, metal, paper, plastic)
- `uniqueID`: `{experiment}_{track_id}`
- `track_length`: Number of frames
- `mean_thermal_intensity_array`: Space-separated thermal intensity values

## Data Generation

Generated by `extract_thermal_data.py` using:
- RGB tracklet masks from `../../tracklets/` (Detectron2 + OC-SORT)
- Homography matrices (`H_XXXX.joblib`) for RGB-to-thermal coordinate warping
- Frame matching CSVs (`frame_matches_*.csv`) for temporal registration
- Thermal video frames from experiment directories

Splits created by `create_splits.py --seed 42` (stratified by class).

## Data Usage Policy
- Train: Model training only
- Validation: Early stopping, hyperparameter tuning (Optuna CV)
- Test: Final evaluation only (never seen during training/tuning)
